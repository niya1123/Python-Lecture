## 基本編

Mac,Winのコマンドプロントを開いて、`python -q`と入力してください。

---

### 算術演算

```python
>>> 1 + 1 # 加算
2
>>> 1 - 1 # 減算
0
>>> 2 * 2 # 乗算
4
>>> 4 / 2 # 除算
2
>>> 4 * 2 + 3
11
>>> 4 * (2 + 3) # ()が優先される
20
>>> 2 ** 3 # べき乗
8
>>> 11 % 4 # 剰余
3
>>> 10 // 3 # 商
3
```

---

### ビット演算

```python
>>> 0 & 1 # AND演算
0
>>> 0 | 1 # OR演算
1
>>> 0 ^ 1 # XOR演算
1
>>> ~0 # NOT演算
1
>>> 1 << 2 # 左ビットシフト
4
>>> 2 >> 1 # 右ビットシフト
1
```

---

### 数値の表記

```python
>>> 5.0 - 1.42 # 浮動小数点の演算
3.58
>>> 1e-4 + 1e-3 # 指数表記(0.0001 + 0.001)
0.0011
>>> 0b10 * 0b11 # 2進表記
6
>>> 0o10 * 0o11 # 8進表記
72
>>> 0x0f / 0x05 # 16進表記
5
```

---

### 変数

```python
>>> x = 2
>>> x
2
>>> y = 2 * 2
>>> y
4
>>> x + y
6
>>> _ * x + y # _は直前の値を格納する特殊な変数.今回は6
16
```

---

### 文字列

```python
>>> 'Hello'
'Hello'
>>> "Hello"
'Hello' # 出力はシングルに統一されているが、違いはない
>>> '2'
'2'
>>> hello = 'Hello'
>>> hello
'Hello'
>>> '\'' # バックスラッシュで特殊文字もただの文字列と判断される
"'"
```

---

### リスト

```python
>>> x = ['A', 'B', 'C']
>>> x
['A', 'B', 'C']
>>> x[0]
'A'
# javaなどは先頭からインデックスを0, 1, 2, …と数えるが、
# pythonはそのほかに一番最後から-1, -2, -3, …と数えることも可能
>>> x[-1]
'C'
>>> x[-2]
'B'
>>> y = ['A', 1, 'B', 2]
>>> y[2]
'B'
# 以下はスライス記法について
>>> y[1:3] # 1 <= x < 3の範囲
[1, 'B']
>>> y[:2] # 指定なしの場合 0 <= x <= 終了インデックスまで
['A', 1]
>>> y[::2] # 先頭から2個飛ばしで値を取得
['A', 'B']
>>> y[::-1] # 後ろから-1飛ばしで値を取得 = リストの逆順
[2, 'B', 1, 'A']
```
